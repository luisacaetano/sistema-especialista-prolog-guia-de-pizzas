# Sistema Especialista em Prolog: Guia de Pizzas

Sistema especialista desenvolvido em Prolog para recomendação de pizzas, considerando preferências do usuário, restrições alimentares e harmonização com bebidas.

## Sobre o Projeto

Trabalho prático da disciplina de Paradigmas de Linguagens de Programação do curso de Ciência da Computação - IFMG Câmpus Formiga.

## Funcionalidades

- Busca de pizzas por nome
- Listagem por categoria (tradicional, especial, vegana, doce, fitness)
- Busca por ingrediente
- Recomendações personalizadas (vegetarianos, crianças, festas, dieta)
- Harmonização com bebidas
- Comparação entre pizzas
- Estatísticas do cardápio

## Como Executar

1. Abra o SWI-Prolog
2. Carregue o arquivo:
```prolog
?- consult('pizzas.pl').
```
3. Inicie o sistema:
```prolog
?- iniciar.
```

## Estrutura

- `pizzas.pl` - Código fonte do sistema
- `Enunciado.pdf` - Enunciado do trabalho
- `SISTEMA ESPECIALISTA EM PROLOG - GUIA DE PIZZA.pdf` - Relatório do trabalho

## Base de Conhecimento
